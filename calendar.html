Journey Calendar System

A comprehensive time management system designed for Journey, Cambio Labs' gamified learning platform, to help instructors and students organize lessons, deadlines, and meetings in a visual, intuitive way.

View Instructor View View Student View

Project Overview

Journey by Cambio Labs is an innovative gamified learning platform that transforms education through engaging elements like points systems, achievements, and interactive competition. The platform has successfully enhanced learning experiences across Brooklyn, the Bronx, Manhattan, and San Francisco. While Journey excels in gamification and student engagement, there's an opportunity to further enhance the platform with comprehensive time management capabilities. This calendar system builds upon Journey's strong foundation by introducing visual time management tools that seamlessly integrate with the existing gamified learning environment.

Key Features

  • Calendar views with monthly and weekly formats featuring color coded events and assignments
  • Upcoming deadlines list organized by priority and urgency
  • Meeting creation and scheduling tools designed specifically for instructors
  • Automated lesson lock and unlock date system for structured progression
  • Gamified progress tracking system with reward mechanisms for student engagement

Identifying the Problem

Current System Limitations

  • Journey lacks a time and deadline system for assignments, making progress tracking challenging
  • Students have no visibility into instructor created class schedules, creating communication gaps
  • No visual calendar interface exists for class meeting times despite having attendance tracking
  • The current repeat system for class dates creates scheduling difficulties for instructors

User Pain Points

  • Students struggle with time management without visual deadline tracking capabilities
  • Instructors require external tools to coordinate lesson timing and monitor student progress
  • No mechanism exists to connect lessons to specific times and dates for structured learning
  • Limited flexibility between in-person, remote, and asynchronous learning modes
